# Architecture Decision Records (internal)

Internal architecture decisions for contributors building **on** this repo — why the
`patterns` CLI is shaped the way it is. These are **not** user-facing and **not** part of any
pattern bundle. (A pattern bundle has its own `adrs/` section — the "justify" docs an author
writes for *their* pattern; that is a different thing from these.)

Numbered `NNNN-slug.md`, append-only. Do not reference these from the README, the docs site,
or the `extract` skill.
